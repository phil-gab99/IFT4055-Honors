Commit
- child-of (except initial commit)
- touches models/model snapshots
- timestamp
- hash (i.e. id)

Model
- connected to one or more commit nodes - relationship specified type of change
  - file not modified within commit - no relationship between nodes
- modelid (i.e. id)

ModelSnapshot
- model at the point in time of a certain commit
- path attribute - model file path at point in time of a commit
